# tapTrack - 时间追踪应用

一款简洁的离线可用时间追踪 Web 应用，浏览器打开即用。

---

## 💡 快速开始

1. **下载后解压**（或直接保存 HTML 文件）
2. **双击 `taptrack.html`** 用浏览器（Chrome/Edge/Firefox/Safari）打开
3. 点击 **签到** 开始计时
4. 点击 **签退** 结束计时，可添加备注
5. 使用 **导出** 功能备份数据

---

## 📋 使用手册

### 基础功能

| 按钮 | 说明 |
|------|------|
| 签到 | 开始记录时间 |
| 签退 | 结束计时，可选择是否扣除休息时间 |
| 手动调整 | 勾选"扣除休息"可输入休息时长自动扣除 |

### 图表统计

- **日报表**：显示当天签到/退时间及工作时长
- **趋势图**：过去 14 天的每日工作时长折线图
- **分布图**：早/午/晚各时段工作分布饼图
- **周均值**：当天在一周内的平均工作时长

### 设置功能

- 🌙 **深色/浅色模式**：点击左上角月亮/太阳图标切换
- 🌐 **中/英文切换**：点击顶部语言选择器
- 🎨 **主题色更换**：点击颜色选择器自定义主色调

### 数据管理

- **导出**：将数据导出为 Markdown 格式文件（.md）
- **导入**：从之前导出的 .md 文件恢复数据
- **自动保存**：数据保存在浏览器本地存储（localStorage）

## ⚠️ 常见问题

**Q: 双击打不开？**
> 确保选择"用浏览器打开"，不要用记事本打开

**Q: 数据会丢失吗？**
> 不会，数据保存在浏览器 localStorage。但如果清除浏览器缓存或换浏览器，数据需要用导出功能备份

**Q: 可以在手机用吗？**
> 可以！用浏览器打开 HTML 文件即可，支持响应式布局

**Q: 换电脑后数据还在吗？**
> 不在。需使用"导出"功能备份 .md 文件，在新电脑用"导入"恢复
